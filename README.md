# MAS-Predictor

Мультиагентная система прогнозирования технического состояния машины на основе Actor Model.

Курсовая работа, ВГУ, факультет ПММ, кафедра МО ЭВМ, 2026.

## О проекте

Система моделирует износ 7 компонентов машины с помощью 7 автономных агентов, работающих параллельно в отдельных потоках. Агенты обмениваются данными через 8 перекрёстных связей и выдают прогноз остаточного ресурса каждого компонента в часах.

## Возможности

- **7 агентов:** двигатель, трансмиссия, тормозная система, подвеска, система охлаждения, топливная система, шины
- **Actor Model:** Idle → Waiting → Ready → Working → Done
- **8 перекрёстных связей** между агентами
- **Параллельное выполнение:** std::jthread (C++20)
- **Коммуникация:** MessageBroker + Blackboard
- **Интерактивная визуализация:** ImGui + ImPlot

## Входные параметры

| Параметр | Диапазон |
|----------|----------|
| Нагрузка | 10–100% |
| Температура | -10..+45°C |
| Тип дороги | Трасса / Город / Гравийка / Бездорожье |
| Переключений/ч | 1–30 |
| Торможений/ч | 1–30 |

## Выходные параметры (20 показателей)

Остаточный ресурс (часы): поршневые кольца, цилиндры, моторное масло, фрикционы, шестерни, масло КПП, колодки, диски, амортизаторы, сайлентблоки, пружины, шины, помпа, ОЖ, топливный фильтр, форсунки.

Также: мощность (кВт), расход топлива (л/ч), температура ОЖ (°C), КПД топливной системы (%).

Итог: ближайшее ТО (часы).
